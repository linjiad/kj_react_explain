### 属性设置

| 属性名称 | 属性类型 | 属性描述 |
| ------ | ------ | ------ |
| leftList | array | 左侧框展示数据 |
| rightList | array | 右侧框展示数据 |
| onChange | function | 回调方法 |

&nbsp;
### 默认穿梭框
穿梭选择框用直观的方式在两栏中移动元素，完成选择行为。
