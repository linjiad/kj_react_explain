## MessageBox 弹框
#### 模拟系统的消息提示框而实现的一套模态对话框组件，用于消息提示、确认消息和提交内容。
&nbsp;
&nbsp;
&nbsp;
### 引用
>const { Modal,Button} = window.common;

&nbsp;
&nbsp;

### 属性设置

| 属性参数 | 属性类型 | 属性描述 |
| ------ | ------ | ------ |
| display  | boolean | 是否显示 |
| width  | int | 提示框宽度 |
| height  | int | 提示框高度 |
| title  | string | 提示框标题 |
| closeModel  | function | 关闭弹出框回调方法 |
