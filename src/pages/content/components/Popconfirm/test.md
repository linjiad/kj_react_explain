## Popconfirm 气泡确认框 
#### 在目标元素附近弹出浮层提示，询问用户
&nbsp;
&nbsp;
&nbsp;
### 引用
>const { Popconfirm } = window.common;

&nbsp;
&nbsp;

### 属性设置

| 属性参数 | 属性类型 | 属性描述 |
| ------ | ------ | ------ |
| title  | string | 提示语 |
| cancelText  | string | 取消按钮提示 |
| okText  | string | 确定按钮文字 |
| onConfirm  | function | 点击确定回调函数 |
