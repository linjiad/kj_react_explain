## Input 输入框
#### 通过鼠标或键盘输入内容，是最基础的表单域的包装。
&nbsp;
&nbsp;
&nbsp;
### 引用
>const { Input } = window.common;

&nbsp;
&nbsp;

### 属性设置

| 属性名称 | 属性类型 | 属性描述 |
| ------ | ------ | ------ |
| placeholder | string | 输入框默认信息 |
| value | string | 绑定的input值 |
| title | string | 输入框标题名称 |
| type | string | 输入框类型|
| onChange | function | 输入框值改变调用方法|
