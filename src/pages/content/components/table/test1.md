## Table 表格 
#### 用于展示多条结构类似的数据，可对数据进行自定义操作
&nbsp;
&nbsp;
&nbsp;
### 引用
>const { Table } = window.common;

&nbsp;
&nbsp;

### 属性设置

| 属性参数 | 属性类型 | 属性描述 |
| ------ | ------ | ------ |
| date  | array | 显示的数据 |
| columns  | array | 显示的数据 |
| pageNum  | int | 当前页 |
| pageSize  | int | 每页多少条 |
