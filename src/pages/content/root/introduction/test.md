# 简介

* 此模板为react通用模板，满足市场上大多数需求。
* 此模板为前端中台解决方案，web方向的react通用框架
* 通过此模板框架+UI组件+基础服务，您可以快速，简洁的开发一套前端项目
* 此模板主旨为您节约开发周期，您只需要关注聚焦业务，底层实现以及定制化开发交给我们
* 通过简单培训，后端人员也可以快速上手前端开发
* 另外我们提供了一套基于此模板的Axure的元件库，方便业务人员与开发之间的沟通
* 此模板源在私服，还请您切换源

# 内容
框架集成了以下几个功能

* 基于react的优化框架
* 基于react的UI组件
* ajax请求插件
* 数据本地化
* 单元测试+集成测试
* 多环境打包
* 路由及嵌套路由配置
